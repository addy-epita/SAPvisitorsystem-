---
name: compound
description: Document learnings to make future work easier
---

Use this command after completing significant work to capture learnings.

1. **Reflect on the work** - What was done? What challenges were encountered?
2. **Identify patterns** - What reusable patterns or utilities emerged?
3. **Document learnings** - Create a file in `docs/learnings/` with:
   - Date and context
   - Problem solved
   - Approach taken
   - Key insights
   - Code snippets (if applicable)
4. **Update CLAUDE.md** - Add any project-wide patterns to CLAUDE.md
5. **Celebrate compounding** - Each documented learning makes future work easier

File naming: `docs/learnings/YYYY-MM-DD-{brief-description}.md`
